# Metodika C# pro děti ze základních škol

Vítejte v repozitáři metodiky C# určené pro děti ze základních škol! Tento repozitář nabízí strukturovaný přístup k výuce C# prostřednictvím interaktivních lekcí a praktických projektů. Cílem je naučit děti základy programování zábavným, poutavým a edukativním způsobem.

## Struktura repozitáře

Tento repozitář je organizován následovně:

### 1. `/cheatsheets`
   - Rychlé referenční příručky pro studenty, které mohou používat během práce na lekcích.
   - Rozděleno podle úrovně lekce (L1, L2, L3, L4, L5).

### 2. `/methodologies`
   - Podrobné plány lekcí a metodiky pro výuku C#.
   - Obsahuje soubory pro metodiku L1 a L2.

### 3. `/preparation`
   - Soubory pro přípravu výuky.
   - Zahrnuje pokyny k instalaci a úvod do metodiky.

### 4. **projects**
   - Série studentských projektů pro každou úroveň lekce.
   - Projekty navržené tak, aby studenty naučily aplikovat to, co se naučili v reálném programování.
   - Rozděleno podle úrovně lekce (L1 až L5).

### 5. **resources**
   - Další materiály, jako jsou odznaky a vizuální pomůcky.
